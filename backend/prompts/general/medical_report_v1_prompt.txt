- Identify and assign the appropriate category based on the test type.

***Prescription***(one block per medicine)
The "prescription" key is an array containing medications prescribed by the doctor. Each object in this array should be filled based on the following rules:
- Medicine Name: This is the name of the medication prescribed. It should be prescribed by the doctor.
- Frequency: Indicates how many times the medicine should be taken and at what times in a day. Format should be the doctor prescribes medicine for morning, set the **Frequency** field to `"1 0 0"`, for afternoon, set the **Frequency** field to `"0 1 0"`, for night, set the **Frequency** field to `"0 0 1"`, for morning and night, set the **Frequency** field to `"1 0 1"`, for morning, afternoon, and night, set the **Frequency** field to `"1 1 1"`.
- If the doctor does **not mention the frequency**, leave the **"Frequency"** field **empty**.
- Dosage: The strength of the medicine (e.g., 500mg), typically mentioned alongside the medicine.
- Duration count and duration_limit: Indicates how long the medicine should be taken. For example, "take this for 3 days" would be duration_count = 3, duration_limit = "days". If not mentioned directly, infer from the transcript.
- Quantity: The total number of units to be consumed over the given period. Calculated as (sum of frequency) × duration_count. For example, frequency 1-0-1 over 5 days → (2 tablets/day) × 5 = 10 tablets.
- When: Specifies the timing or condition under which the medicine should be taken. This can include phrases like "after meals", "before meals", "in the morning", "at night", or "on an empty stomach". Also capture conditional instructions such as "take only when you have a headache" or "use during allergy symptoms". Include all applicable timing cues or conditions mentioned by the doctor in the transcript.
- Remarks: Any additional instruction or advice about the medication said by doctor. Example:"take with water", "only when pain is severe", etc.
